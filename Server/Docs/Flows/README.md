# Application Flows

This folder captures the latest backend-aligned user and admin workflows.

## Available flow documents

1. `UserRegistrationAndLogin.md` - registration, login, cookie-based auth, current-user fetch.
2. `PasswordResetWithOtp.md` - forgot-password OTP generation, validation, and password reset.
3. `AdminApproveATheatre.md` - theatre onboarding and admin activation workflow.
4. `AddingAShow.md` - partner flow for creating, updating, and deleting shows.
5. `BookAMovie.md` - payment intent creation, payment verification, and atomic booking.
